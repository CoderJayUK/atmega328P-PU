# ATMega328P-PU DIP-28

[Datasheet](Microchip-ATMEGA328P-PU-datasheet.pdf)

## Specs

* **AVR Microcontroller**
* **Supply voltage**: 1.8V - 5.5V
* **Core Size**: 8-bit
* **Speed:** up to 20MHz
* **I/O Pins**: 23
* **Program memory size**: 32Kb (16K x 16)
* **Program memory type**: Flash
* **EEPROM size**: 1K * 8
* **RAM size**: 2K * 8
* **Package**: DIP-28
